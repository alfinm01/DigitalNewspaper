## Clone setup (on Windows example)

``` bash
# install environments
$ py -3 -m venv venv
$ venv\Scripts\activate
(venv) $ pip install -r requirements.txt
```

## Running locally (on Windows example)

``` bash
# serve at default https://localhost:5000
$ venv\Scripts\activate
(venv) $ py main.py

# to leave venv
(venv) $ deactivate
```
